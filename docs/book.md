# Вступ

Цей навчальний посібник розроблено як експериментальний фрагмент спецкурсу, спрямованого на формування професійної готовності майбутніх учителів математики до відповідального, методично обґрунтованого та критичного використання технологій штучного інтелекту.

Посібник поєднує інструментальний, методичний та критично-аналітичний виміри інтеграції ШІ в освітній процес та орієнтований на український контекст професійної підготовки вчителя.
